# Source Fidelity

Every non-trivial deck must preserve evidence boundaries.

- Use `source_anchors` for Markdown headings, PDF pages, figure/table IDs, code output paths, prior slide numbers, or reviewer comments.
- Separate completed results, planned work, hypotheses, and speculation.
- Keep formulas in LaTeX.
- Do not invent claims, citations, result values, or decisions.
- Keep the deck plan alongside the delivered deck.
