# Statistical Decision Notes

- Define the unit of analysis before selecting a test.
- Match the model to outcome scale, sampling design, repeated measures, and missingness.
- Prefer effect sizes and uncertainty intervals over significance-only reporting.
- Use robust or permutation/bootstrap alternatives when assumptions are questionable and sample size permits.
- Clearly label exploratory analyses and multiplicity risks.
- For causal language, require design support or state assumptions explicitly.
