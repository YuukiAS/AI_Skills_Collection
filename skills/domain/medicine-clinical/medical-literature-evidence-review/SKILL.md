# Medical Literature Evidence Review

## Workflow

1. Define PICO, condition, population, intervention/exposure, comparator, outcomes, time horizon, and jurisdiction.
2. Search or verify current authoritative sources before making medical claims.
3. Rank evidence: guidelines/regulators, systematic reviews, randomized trials, observational studies, mechanistic or expert opinion.
4. Extract effect sizes, harms, uncertainty, population fit, and date/source boundaries.
5. Write a concise evidence summary with limitations and clinician-review language.

## References

- Read `references/source-notes.md` for source hierarchy and citation boundaries.
- Read `references/evidence-table-template.md` when building a structured evidence table.
