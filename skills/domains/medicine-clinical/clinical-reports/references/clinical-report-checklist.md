# Clinical Report Checklist

- Classify report type before drafting.
- Remove HIPAA identifiers or equivalent local identifiers unless the user explicitly provides de-identified data.
- For case reports, check CARE and journal instructions.
- For trial reports, check CONSORT, ICH-E3, SAE/CSR requirements, and the protocol.
- For diagnostic reports, distinguish measurements/findings from interpretation.
- For patient documentation, avoid unsupported diagnosis, medication, or causality claims.
- Include consent/review/sign-off language where required.
