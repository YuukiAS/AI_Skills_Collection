# Source Notes

This skill synthesizes writing guidance from the deep research PDF in `reference/面向科研组会 PDF 的写作 skill 与工作流深度研究.pdf` and from these external sources cited there:

- Gopen and Swan, "The Science of Scientific Writing": information flow, old-to-new structure, and sentence emphasis.
- Caltech Scientific Storytelling handout: scientific communication as known/unknown and problem/solution movement.
- NIH plain language guidance: write for audience, purpose, core message, and reader questions.
- UNC Writing Center, Figures and Charts: captions should be concise, complete, and point readers to the key feature.
- PLOS figure guidance: figure legends should define panels, symbols, and abbreviations without becoming a methods section.
- Google developer documentation style guidance: direct, natural, respectful technical prose; active voice where responsibility matters.
- Vale (`vale-cli/vale`, checked locally at commit `5242b459`, MIT): treated as an implementation reference for programmable prose linting, not vendored or required at runtime.

No external repository is vendored into this skill. The skill is a local synthesis and should be maintained as a concise workflow.
