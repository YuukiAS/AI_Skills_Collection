# Bioinformatics Analysis Taxonomy

Reviewed for v3.6.0 from a user-supplied screenshot of the private Notion page
`常用生信分析流程Skill`. The original post required payment, so this repository
does not copy or reconstruct the paid workflow content.

Use only the visible category labels as a routing taxonomy:

| Visible category | Routing note |
|---|---|
| ATAC-seq chromatin accessibility | Route to ATAC/scATAC-capable workflows such as deepTools, geniml, gtars, scvi-tools PeakVI, or project-specific ATAC pipelines. |
| Biofigure polish | Route to figure polishing, visual QA, or frontend/visualization skills; this is not a biological analysis method. |
| bulk RNA differential analysis | Route to `pydeseq2` or established bulk RNA-seq differential expression workflows. |
| CRISPR screen analysis | Route to DepMap or project-specific CRISPR screen analysis; verify assay type and score model before use. |
| GO/KEGG/GSEA enrichment | Route to enrichment-capable providers and tools such as gget Enrichr, KEGG, Reactome, STRING, or bioservices. |
| Immune repertoire analysis | Route to project-specific repertoire/TCR/BCR tooling; no active general skill is vendored here yet. |
| Microbiome analysis | Route to `scikit-bio` and community ecology workflows. |
| Network and pathway analysis | Route to STRING, Reactome, KEGG/bioservices, arboreto, or NetworkX-style graph analysis depending on input type. |

Do not claim this taxonomy validates any paid post content. Treat it as a
high-level folder classification only.
