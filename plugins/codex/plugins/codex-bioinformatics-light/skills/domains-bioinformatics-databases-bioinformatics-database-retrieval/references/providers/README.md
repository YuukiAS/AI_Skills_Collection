# Provider Notes

This directory contains lightweight routing notes for database providers that
used to be represented as one skill each. Keep provider-specific API examples
here so the active skill list stays small.

Covered providers include UniProt, KEGG, Reactome, GEO, GWAS Catalog,
OpenTargets, AlphaFold DB, Ensembl, gnomAD, GTEx, STRING, JASPAR, InterPro,
Monarch, ENA, DepMap, CELLxGENE Census, and NCBI Gene/gget.
