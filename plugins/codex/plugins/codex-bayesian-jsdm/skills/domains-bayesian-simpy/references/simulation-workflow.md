# Simulation Workflow Notes

- Keep process logic separate from parameters and random distributions.
- Validate with a deterministic toy model before stochastic scenarios.
- Record seed strategy, warm-up period, run length, replications, and confidence intervals.
- Monitor queues, resource utilization, wait times, throughput, and dropped/reneged entities as appropriate.
- Use scenario tables for policy comparisons; do not over-interpret a single trajectory.
