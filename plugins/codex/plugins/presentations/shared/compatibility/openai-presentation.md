# Official Presentation/Slides Compatibility

This repository supplies deck planning, source fidelity, templates, and QA rules.

The official Presentation/Slides capability should handle:

- creating and editing `.pptx` or Google Slides files;
- manipulating shapes, text, images, charts, and layouts;
- exporting to PDF or images for QA;
- inspecting existing deck structure.

If the capability is unavailable, mark any fallback explicitly and avoid maintaining a second equivalent engine in this repository.
