# Visual QA

Render the deck before delivery.

- Export PPTX/Slides to PDF or slide images.
- Inspect title, section navigation, page numbers, figures, equations, text wrapping, contrast, and alignment.
- Check that the deck is editable when `.pptx` was requested.
- Do not use PDF page images as slide backgrounds unless explicitly delivering image/PDF slides.
- Re-render changed slides after fixes.
